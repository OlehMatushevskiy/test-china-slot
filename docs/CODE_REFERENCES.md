# Code references

- `src/configs/GameAssets.ts`: canonical asset imports and Pixi aliases.
- `src/core/Game.ts`: persistent sound preference and looping background-music playback.
- `src/blocks/SoundToggleBlock.ts`: player sound opt-in; begins background music after selecting ON.
- `src/blocks/BackgroundBlock.ts`: scales the nature background to fill the viewport.
- `src/blocks/SlotBoardBlock.ts`: renders the Chinese board and the three fortune symbols.
- `src/scenes/MainGameScene.ts`: composes the active game scene.
- `assets/chinese-slot-background.png`: nature backdrop.
- `assets/chinese-slot-board.png`: colorful red/jade/gold board.
- `assets/symbol-lucky-dragon.png`, `assets/symbol-red-lantern.png`, `assets/symbol-golden-ingot.png`: active symbols.
