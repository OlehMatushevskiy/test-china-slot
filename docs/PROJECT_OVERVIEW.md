# Project overview

Chinese-style iGaming slot prototype built with React, TypeScript, PixiJS, Tailwind CSS, and Webpack. The main scene layers a nature background behind an ornate red, jade, and gold slot board with three fortune symbols.

Run with `npm run dev`; validate with `npm run build`. Visual assets are in `assets/` and aliases are centralized in `src/configs/GameAssets.ts`. `audio/background-music.mp3` is looping background music, started from the player's sound opt-in choice.
